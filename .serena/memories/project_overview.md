# AI Course Platform - Project Overview

## Purpose
B2B English training course generation platform with AI-powered SOP integration and CEFR alignment. The platform enables sales teams to create customized English training courses for business clients.

## Tech Stack
- **Frontend**: Next.js 14 with TypeScript, React 18
- **UI Framework**: Tailwind CSS with shadcn/ui components
- **AI Integration**: OpenAI API, CopilotKit for AI chat interfaces
- **Agent System**: BMAD (Business Multi-Agent Delegation) system
- **Backend**: Python FastAPI (backend directory)
- **Database**: Supabase (configured but not actively used in current implementation)
- **Authentication**: JWT-based with role-based access control
- **Error Tracking**: Sentry integration
- **Deployment**: Ready for Vercel, Netlify, AWS Amplify

## Key Features
1. **AI-Powered Course Generation**: Creates CEFR-aligned English training courses
2. **SOP Integration**: Upload and analyze Standard Operating Procedures
3. **CopilotKit AI Assistant**: Intelligent chat interface for training expertise
4. **B2B Sales Portal**: Complete workflow management for training requests
5. **CEFR Compliance**: Validated alignment with Common European Framework
6. **BMAD Agent System**: Multi-agent architecture for complex AI operations
