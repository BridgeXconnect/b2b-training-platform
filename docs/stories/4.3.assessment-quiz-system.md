# Story 4.3: Assessment & Quiz System

## Status
Draft

## Story
**As a** language learner,
**I want** to take AI-generated assessments and quizzes aligned with my CEFR level,
**so that** I can measure my progress, receive immediate feedback, and have the difficulty adapt to my performance.

## Acceptance Criteria
1. AI-generated assessments based on course content
2. CEFR-aligned quiz creation with appropriate difficulty levels
3. Automated scoring and feedback system
4. Performance analytics for assessments with detailed insights
5. Adaptive difficulty based on performance history
6. Assessment history and review functionality
7. Integration with existing progress tracking system
8. Support for multiple question types (multiple choice, fill-in-blank, essay, listening, speaking, ordering)
9. Real-time feedback and explanations for incorrect answers
10. Export assessment results and analytics

**New Assessment Models Required:**
```typescript
// Assessment question types and interfaces
export type QuestionType = 'multiple-choice' | 'fill-blank' | 'essay' | 'listening' | 'speaking' | 'ordering';

export interface AssessmentQuestion {
  id: string;
  type: QuestionType;
  question: string;
  context?: string; // Business context
  options?: string[]; // For multiple choice
  correctAnswer: string | string[];
  explanation?: string;
  cefrLevel: string;
  difficulty: number; // 1-10 scale
  category: 'speaking' | 'writing' | 'listening' | 'reading';
}

export interface AssessmentSession {
  id: string;
  userId: string;
  questions: AssessmentQuestion[];
  currentQuestionIndex: number;
  answers: Record<string, any>;
  startTime: Date;
  endTime?: Date;
  isCompleted: boolean;
}

export interface AssessmentResult {
  id: string;
  sessionId: string;
  userId: string;
  score: number;
  totalQuestions: number;
  correctAnswers: number;
  timeSpent: number;
  cefrLevel: string;
  skillBreakdown: Record<string, number>;
  feedback: string[];
  recommendations: string[];
  completedAt: Date;
}
```

```typescript
// Assessment generation API
POST /api/assessments/generate
{
  cefrLevel: string;
  questionTypes: QuestionType[];
  businessContext: string;
  difficulty: number;
  questionCount: number;
}

// Assessment submission API
POST /api/assessments/submit

  sessionId: string;

  timeSpent: number;
}

GET /api/assessments/results/:sessionId

```
