## Story
**As a** language learner,
**I want** to take AI-generated assessments and quizzes aligned with my CEFR level,
**so that** I can measure my progress, receive immediate feedback, and have the difficulty adapt to my performance.

## Acceptance Criteria

### Core Functionality
1. **AI-Generated Assessments**: Generate 10-20 questions based on course content with business context
2. **CEFR Alignment**: Questions automatically match user's current CEFR level (A1-C2) with appropriate vocabulary and complexity
3. **Multiple Question Types**: Support multiple choice, fill-in-blank, essay, listening, speaking, and ordering questions
4. **Automated Scoring**: Immediate scoring with percentage and CEFR-appropriate feedback

### Business Rules
5. **Adaptive Difficulty**: After 3 consecutive wrong answers, reduce difficulty one CEFR sub-level; after 5 consecutive correct answers, increase difficulty
6. **Assessment Length**: Default 15 questions, user can choose 10-30 questions
7. **Time Limits**: Optional timer (5-60 minutes), with auto-submit when time expires
8. **Retry Policy**: Users can retake assessments after 24 hours, keeping best score

### User Experience  
9. **Real-time Feedback**: Show correct answers and explanations immediately after each question
10. **Progress Tracking**: Visual progress bar showing completion percentage and current performance
11. **Assessment History**: View past assessments with scores, dates, and improvement trends
12. **Export Results**: Download assessment results as PDF or CSV for personal records

### Integration Requirements
13. **Progress Dashboard**: Assessment scores automatically update user's CEFR progress tracking
14. **Course Content**: Assessments reference specific course modules and business terminology
15. **Performance Analytics**: Detailed insights on strengths/weaknesses by skill area

```typescript
// Assessment question types and interfaces
export type QuestionType = 'multiple-choice' | 'fill-blank' | 'essay' | 'listening' | 'speaking' | 'ordering';

export interface AssessmentQuestion {
  id: string;
  type: QuestionType;
  question: string;
  context?: string; // Business context
  options?: string[]; // For multiple choice
  correctAnswer: string | string[];
  explanation?: string;
  cefrLevel: string;
  difficulty: number; // 1-10 scale
  category: 'speaking' | 'writing' | 'listening' | 'reading';
}

export interface AssessmentSession {
  id: string;
  userId: string;
  questions: AssessmentQuestion[];
  currentQuestionIndex: number;
  answers: Record<string, any>;
  startTime: Date;
  endTime?: Date;
  isCompleted: boolean;
}

export interface AssessmentResult {
  id: string;
  sessionId: string;
  userId: string;
  score: number;
  totalQuestions: number;
  correctAnswers: number;
  timeSpent: number;
  cefrLevel: string;
  skillBreakdown: Record<string, number>;
  feedback: string[];
  recommendations: string[];
  completedAt: Date;
}
```
