# Story 4.4: User Profile & Preferences

## Status
Draft

## Story
**As a** language learner,
**I want** to manage my profile, learning preferences, and CEFR level settings,
**so that** I can personalize my learning experience and track my progression accurately.

## Acceptance Criteria
1. User profile management interface with editable personal information
2. Learning preferences configuration (goals, study schedule, preferred topics)
3. CEFR level tracking and manual updates
4. Personalization settings for AI interactions and content recommendations
5. Profile sharing capabilities with privacy controls
6. Integration with progress tracking and assessment systems
7. Export user data and learning history
8. Notification preferences and communication settings
9. Accessibility and language preferences
10. Profile completion tracking and recommendations

### Data Models
**Source: components/learning/ProgressDashboard.tsx**
- Existing user metrics: `ProgressMetrics`, `LearningGoal`, `Achievement`
- CEFR tracking: `cefrProgress: { current: string, nextLevel: string, progressToNext: number }`

**New Profile Models Required:**
```typescript
export interface UserProfile {
  id: string;
  userId: string;
  personalInfo: {
    firstName: string;
    lastName: string;
    email: string;
    avatar?: string;
    bio?: string;
    location?: string;
    timezone: string;
  };
  learningPreferences: {
    goals: string[];
    studySchedule: {
      preferredDays: string[];
      preferredTime: string;
      sessionDuration: number; // minutes
    };
    businessContext: string;
    industry: string;
    topics: string[];
    learningStyle: 'visual' | 'auditory' | 'kinesthetic' | 'mixed';
    pace: 'slow' | 'moderate' | 'fast';
    challengeLevel: 'comfortable' | 'challenging' | 'very-challenging';
  };
  cefrTracking: {
    currentLevel: string;
    targetLevel: string;
    progressionHistory: Array<{
      level: string;
      achievedAt: Date;
      method: 'assessment' | 'manual' | 'auto';
    }>;
    lastUpdated: Date;
  };
  preferences: {
    notifications: {
      email: boolean;
      inApp: boolean;
      reminders: boolean;
      achievements: boolean;
      marketing: boolean;
    };
    communication: {
      aiPersonality: 'formal' | 'casual' | 'encouraging' | 'challenging';
      feedbackFrequency: 'immediate' | 'daily' | 'weekly';
      language: string;
    };
    accessibility: {
      fontSize: 'small' | 'medium' | 'large';
      colorScheme: 'light' | 'dark' | 'auto';
      screenReader: boolean;
    };
  };
  privacy: {
    profileVisibility: 'public' | 'private' | 'friends';
    shareProgress: boolean;
    shareAchievements: boolean;
    dataRetention: '30days' | '90days' | '1year' | 'indefinite';
  };
  profileCompletion: {
    score: number; // 0-100
    completedSections: string[];
    recommendations: string[];
  };
}
```
